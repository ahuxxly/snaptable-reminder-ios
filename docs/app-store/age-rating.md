# Age Rating Draft

This is a working draft for the App Store age rating questionnaire. The final rating is determined by Apple's questionnaire.

## Expected Rating

Likely 4+ for version 1, subject to Apple's final questionnaire.

## Content Notes

Version 1 does not include:

- objectionable content;
- user-generated public feeds;
- unrestricted web access;
- gambling;
- contests;
- alcohol, tobacco, or drug references;
- medical treatment advice;
- financial or investment advice;
- violence, horror, sexual content, or profanity;
- location sharing;
- social networking.

## User-Entered Content

Users can enter private local text records and OCR results. The app does not publish, share, moderate, or host user-generated content.

## Re-check Required If Added Later

Revisit age rating if a future version adds:

- web browsing;
- shared/community content;
- cloud document templates;
- AI-generated content shown from remote sources;
- medical, financial, or legal analysis.
